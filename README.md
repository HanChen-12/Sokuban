# 推箱子游戏 / Sokoban Game
基于 Unreal Engine 5 蓝图开发的正交3D推箱子游戏。
---
## 功能特性
### 核心玩法
- 正交俯视视角
- WASD / 方向键控制玩家移动
- 推箱子到目标点通关
- 撤回功能（Z键，最多50步）
- 重置关卡（R键）
- ESC 返回主菜单

### 关卡系统
- 6个官方预设关卡（简单到难）
- 自定义关卡编辑器
- 自定义关卡存档（重启游戏后保留）
- 关卡选择界面

### 关卡编辑器
- 网格绘制工具（墙/地板/目标/箱子/玩家/橡皮擦）
- 关卡验证（检查玩家、箱子、目标点数量）
- 自定义关卡名称
- 保存/测试/清空功能
- 调整网格大小

### 界面系统
- 主菜单
- 关卡选择（支持编辑/删除自定义关卡）
- 游戏 HUD（步数显示）
- 通关界面
- 返回主菜单按钮

---

## 环境要求

- Unreal Engine 5.5.4
- Windows 10 / 11
